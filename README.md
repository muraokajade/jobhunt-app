# JobHunt

JobHuntは、転職活動中の応募企業・選考状況・面談予定・次アクションを一元管理する転職活動CRMです。

## コンセプト

5秒で、今の選考状況と次にやることがわかる。

Excelで管理していた転職ログをWebアプリ化し、複数媒体・複数企業への応募状況を素早く整理できることを目的としています。

## 背景

本プロジェクトは、Laravel CRUD復習用に作成したTaskLogを土台に、実体験ベースの転職活動管理アプリ「JobHunt」へ発展させたものです。

## 主な機能予定

- 企業一覧管理
- 企業登録・編集・削除
- 選考ステータス管理
- 応募日・面談日管理
- 次アクション管理
- メモ管理
- 落選段階管理
- 選考ファネル可視化
- Googleカレンダー登録URL生成

## 技術スタック

### Backend

- Laravel
- PHP
- MySQL or SQLite

### Frontend

- React
- TypeScript
- Tailwind CSS

## 開発目的

- Laravel / React / TypeScript の実務感覚を戻す
- 応募後の転職活動管理という実課題をWebアプリ化する
- 将来的に販売可能なWebブラウザアプリへ展開する
